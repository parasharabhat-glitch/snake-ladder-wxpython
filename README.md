# Snake & Ladder Game

A simple 2-player Snake & Ladder game built using Python and the wxPython GUI framework.

## Features
- Grid-based game board (10x10)
- Support for 2 players (Red and Blue)
- Snakes and Ladders with distinct colors and emojis
- Automated dice rolls and movement logic
- Win detection and game reset

## Requirements
- Python 3.x
- wxPython

## How to Run
1. Install wxPython: `pip install wxPython`
2. Run the script: `python jackfruit5.py`
